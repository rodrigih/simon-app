How to Play

The object of the game is to copy the sequence of buttons shown by the computer. The sequence starts with one button and increases by one button for each time the player correctly plays the sequence.

The game is won when the player gets the sequence of 20 buttons correct.

Should the player guess the sequence wrong, the sequence is repeated.

Strict Mode

Strict mode has the same rules as above, but if a player guesses wrong, a new sequence of the same length is shown.

Endless Mode

Endless mode has the same rules as the regular game mod,e but it continues endlessly. If the player guesses wrong, the game resets and starts over.
